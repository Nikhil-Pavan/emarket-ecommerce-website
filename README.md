# Emarket E-Commerce Website

Emarket is a responsive e-commerce website built using HTML, CSS, and JavaScript.  
The project simulates a basic online shopping experience including product browsing, cart management, checkout flow, and authentication UI.

## Live Demo

https://nikhil-pavan.github.io/emarket-ecommerce-website/

## Repository

https://github.com/Nikhil-Pavan/emarket-ecommerce-website

## Features

- Product listing page
- Add to cart functionality
- Cart dropdown with quantity controls
- Cart page with dynamic updates
- Checkout system
- Address saving using localStorage
- Login / Signup interface
- Product search feature
- Responsive design for mobile and desktop

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
